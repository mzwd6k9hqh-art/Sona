# Sona Business Helper (simple client-side app)
Files:
- index.html
- style.css
- app.js
- manifest.json
- vercel.json

Deploy:
1. Push repo to GitHub.
2. On Vercel: New Project → Import GitHub repo → Deploy.
3. After deployment copy Production URL (https://your-app.vercel.app).
4. In Whop Developer App settings set Base URL = production URL.
5. Leave Redirect URL empty (no OAuth).
6. Ensure 2FA enabled on Whop account before interacting with Developer features.
